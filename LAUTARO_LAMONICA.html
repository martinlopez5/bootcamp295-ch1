Examen Git y Github

Alumn@: [Nombre Completo]

  1. Que es un sistema de control de versiones?

    [Un sistema de control de versiones es una herramienta utilizada en el desarrollo de software para evitar el riesgo de conflictos que puedan surgir al trabajar en colaboración con otros equipos de desarrollo]

  2. Cual es la diferencia entre git y github?

    [Git sirve para controlar versiones de los archivos, y Github es una plataforma para guardar tus repositorios.]

  3. Cómo creo una rama en git?

    [Con el comando git branch nueva_rama vas a poder crear una nueva rama.]

  4. Cómo clono un repositorio?

    [Ir a la pagina de GitHub donde se encuentra el repo, tiene que aparecer una opcion que dice CODE con una flechita para abajo, desde ahi te va a dejar copiar el link para clonar el repo. Luego desde la Git Bash, con el siguiente comando: $ git clone https://link-del-repo.com/xxxx vas a tener clonado el repo local.]
